Mobility

We are moving towards giving staff the ability to use most of the services out of the office both on mobile devices and on company laptops. I have been assisting in deploying and managing company smartphones, we have been moving on from unsupported windows phones to newer android ones that should be easier to integrate into our systems and create apps for.

Some of the mobile platforms that we have available that I support is the new care system (a tablet app with almost full functionality as the web version, just easier to use on a touch screen) and the highways system (an app for windows tablets that allows workers to keep track of jobs and assign files to jobs while offline to be synced with the server later).

We also have applications that can be accessed remotely from a work laptop from any network from any network. The connection is secured by an encrypted VPN. The applications that allow us to tunnel into the network and run applications in remote instances are Citrix (which is being phased out) and RDS (which will be replacing Citrix). I have taken part in testing RDS for some applications that it is being deployed for.
